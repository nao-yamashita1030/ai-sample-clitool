# 画面詳細設計: 調査結果詳細画面

## 基本情報

- **画面名**: 調査結果詳細画面
- **画面ID**: SCREEN-003
- **URL**: `/results/:scanHistoryId/detail/:resultId`
- **説明**: リンク切れの詳細情報を表示する画面
- **対象ユーザー**: すべてのユーザー

## 画面の目的

- リンク切れの詳細情報を表示する
- リンク元URL、HTTPステータスコード、エラーメッセージなどを表示

## 画面要素

### 要素1: URL表示

- **要素名**: URL表示
- **種類**: テキスト（リンク）
- **説明**: リンク切れのURLを表示（クリックでブラウザで開く）

### 要素2: リンク元URL表示

- **要素名**: リンク元URL表示
- **種類**: テキスト（リンク）
- **説明**: リンク切れが見つかったページのURLを表示

### 要素3: HTTPステータスコード表示

- **要素名**: HTTPステータスコード表示
- **種類**: テキスト（バッジ）
- **説明**: HTTPステータスコードを色分けして表示（404=赤、500=オレンジなど）

### 要素4: エラーメッセージ表示

- **要素名**: エラーメッセージ表示
- **種類**: テキスト
- **説明**: エラーメッセージを表示（存在する場合）

## 画面遷移

### 遷移元

- [調査結果一覧画面](./results-list-screen.md) → この画面（詳細ボタンクリック）

### 遷移先

- この画面 → [調査結果一覧画面](./results-list-screen.md)（戻るボタンクリック）

## 処理フロー

1. 画面表示時
   - `resultId`から調査結果を取得
   - 詳細情報を表示
